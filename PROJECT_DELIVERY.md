🚀 部署和使用
================================================================================

### 快速开始

```bash
# 1. 进入项目目录
cd "/Volumes/MyDisk/App programs/dingtalk-h5-app"

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量
cp .env.example .env
nano .env  # 填写钉钉凭证

# 4. 启动应用
python run.py

# 5. 访问设计器
open http://localhost:8000/designer
```

### Git 同步

```bash
# 使用交互式脚本（推荐）
./sync_to_git_interactive.sh

# 或查看详细指南
cat GIT_SYNC_GUIDE.md
```
