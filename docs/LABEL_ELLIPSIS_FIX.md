问题: 标签控件预览和打印时只能显示"......"

================================================================================
🐛 问题分析
================================================================================

问题根本原因:
---------------------------------------------------------------
1. 标签控件宽度太小
   - 文本内容超出控件宽度
   - overflow_marker = True（默认）
   - 触发省略号逻辑

2. fit_with_ellipsis 函数逻辑
   ```python
   def fit_with_ellipsis(src: str) -> str:
       if not overflow_marker:
           return src
       ell = "..."
       # 尝试添加省略号
       if line_width(src + ell) <= rect.width - 1:
           return src + ell
       # 如果放不下，逐个删除字符
       s = src
       while s:
           s = s[:-1]
           if line_width(s + ell) <= rect.width - 1:
               return s + ell
       # 如果空间很窄，只返回省略号
       return ell if line_width(ell) <= rect.width - 1 else src
   ```

3. 可能的情况
   - 控件宽度 < 文本宽度 → 显示 "文本..."
   - 控件宽度 < "文本..." 宽度 → 显示 "文..."
   - 控件宽度 < "文..." 宽度 → 显示 "..."
   - 控件宽度 < "..." 宽度 → 显示原文本（但可能被截断）

================================================================================
✅ 解决方案
================================================================================

方案 1: 增加标签控件宽度（推荐）
---------------------------------------------------------------
在设计器中调整标签控件的宽度，确保能容纳完整文本。

方案 2: 禁用 overflow_marker
---------------------------------------------------------------
在设计器中取消勾选"溢出标记"，这样文本会直接截断而不显示"..."。

方案 3: 启用 text_wrap
---------------------------------------------------------------
在设计器中启用"文本换行"，让文本自动换行显示。

方案 4: 优化 fit_with_ellipsis 逻辑（代码修复）
---------------------------------------------------------------
修改逻辑，当控件太窄时，优先显示文本而不是省略号。

```python
def fit_with_ellipsis(src: str) -> str:
    if not overflow_marker:
        return src
    ell = "..."
    # 如果完整文本+省略号能放下，返回
    if line_width(src + ell) <= rect.width - 1:
        return src + ell
    # 如果控件太窄，连省略号都放不下，直接返回原文本
    if line_width(ell) > rect.width - 1:
        return src
    # 逐个删除字符，尝试添加省略号
    s = src
    while s:
        s = s[:-1]
        if line_width(s + ell) <= rect.width - 1:
            return s + ell
    # 如果一个字符都放不下，返回省略号
    return ell
```

优化后的逻辑:
1. 如果 "文本..." 能放下 → 返回 "文本..."
2. 如果 "..." 都放不下 → 返回原文本（让 PyMuPDF 自然截断）
3. 如果能放下部分文本 → 返回 "部分文本..."
4. 如果一个字符都放不下 → 返回 "..."
