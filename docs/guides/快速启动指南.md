# 快速启动指南

## 🚀 立即开始

### 1. 启动服务器

```bash
cd "/Volumes/MyDisk/App programs/dingtalk-h5-app"
python3 run.py
```

或使用重启脚本：
```bash
./restart_server.sh
```

打开浏览器访问：
- **主页**: http://localhost:5000
- **管理后台**: http://localhost:5000/admin


## 🔧 常见问题

### 问题 1: 端口被占用

**错误信息**: `Address already in use`

**解决方法**:
```bash
# 查找占用 5000 端口的进程
lsof -i :5000

# 杀死进程
kill -9 <PID>

# 或修改 run.py 使用其他端口
```

### 问题 2: 依赖包缺失

**错误信息**: `ModuleNotFoundError: No module named 'xxx'`

**解决方法**:
```bash
pip3 install -r requirements.txt
```

### 问题 3: 权限问题

**错误信息**: `Permission denied`

**解决方法**:
```bash
chmod +x *.sh
chmod +x *.py
```

### 问题 4: 钉钉 API 调用失败

**检查清单**:
1. 检查 `.env` 文件中的凭证是否正确
2. 确认网络连接正常
3. 查看日志文件了解详细错误

## 📝 配置说明

### 环境变量 (.env)

必需配置：
```bash
DINGTALK_APP_KEY=your_app_key
DINGTALK_APP_SECRET=your_app_secret
DINGTALK_CORP_ID=your_corp_id
FLASK_SECRET_KEY=your_secret_key
```

可选配置：
```bash
PUBLIC_BASE_URL=http://localhost:5000
REQUEST_TIMEOUT_SECONDS=10
```

### 路径配置

所有路径现在自动使用项目根目录，无需手动配置。

如需自定义，可在 `.env` 中设置：
```bash
PDF_TEMPLATE_DIR=/custom/path/to/templates
OUTPUT_DIR=/custom/path/to/outputs
```
