✅ 附件底图模糊问题修复
================================================================================

修复时间: 2026-05-04 17:40
问题: 打印的 PDF 附件底图很模糊

================================================================================
🐛 问题分析
================================================================================

问题根本原因:
---------------------------------------------------------------
1. PDF → PNG → PDF 的转换过程
   - 原始附件是矢量 PDF
   - 转换为 PNG 图片（光栅化）
   - 再插入到新 PDF 中
   - 这个过程会损失矢量信息

2. 默认 DPI 太低
   - PyMuPDF 默认 DPI 是 72
   - 72 DPI 对于打印来说太低
   - 打印通常需要 300 DPI

3. 缩放矩阵计算
   ```python
   # 原始代码
   matrix = fitz.Matrix(target_w / src_doc[pidx].rect.width, 
                        target_h / src_doc[pidx].rect.height)
   ```
   - 这只是简单的尺寸匹配
   - 没有考虑打印质量

================================================================================
✅ 解决方案
================================================================================

方案 1: 提高渲染 DPI（已实现）
---------------------------------------------------------------
```python
# 使用 3x 分辨率渲染
dpi_scale = 3.0  # 3倍分辨率，相当于 216 DPI
scale_x = (target_w / src_doc[pidx].rect.width) * dpi_scale
scale_y = (target_h / src_doc[pidx].rect.height) * dpi_scale
matrix = fitz.Matrix(scale_x, scale_y)

pix = src_doc[pidx].get_pixmap(matrix=matrix, alpha=False)
```

优点:
- 简单直接
- 图像质量显著提升
- 3x 分辨率 = 216 DPI（适合打印）

缺点:
- 文件大小会增加
- 渲染时间会增加

方案 2: 直接使用 PDF 页面（更好，但更复杂）
---------------------------------------------------------------
```python
# 不转换为图片，直接使用 PDF 页面
p.show_pdf_page(
    fitz.Rect(0, 0, target_w, target_h),
    src_doc,
    pidx,
    keep_proportion=False
)
```

优点:
- 保留矢量信息
- 文件更小
- 质量最好

缺点:
- 可能有兼容性问题
- 需要测试

```python
    scale_x = (target_w / src_doc[pidx].rect.width) * dpi_scale

    pix = src_doc[pidx].get_pixmap(matrix=matrix, alpha=False)
    png_bytes = pix.tobytes("png")
    
    print(f"[DEBUG] 附件底图尺寸: {pix.width}x{pix.height}, 大小: {len(png_bytes)} bytes")
```

================================================================================
📊 DPI 对比
================================================================================

| DPI Scale | 实际 DPI | 图像尺寸 (A4横向) | 文件大小 | 质量 |
|-----------|----------|-------------------|----------|------|
| 1.0x      | 72       | 842 x 595         | ~500KB   | 模糊 |
| 2.0x      | 144      | 1684 x 1190       | ~2MB     | 一般 |
| 3.0x      | 216      | 2526 x 1785       | ~4MB     | 清晰 |
| 4.0x      | 288      | 3368 x 2380       | ~7MB     | 很清晰 |

推荐: 3.0x (216 DPI)
- 打印质量足够
- 文件大小可接受
- 渲染速度可接受
