using_attachment_bg = (
    cover_source_mode == "attachment"

)

💡 工作原理
================================================================================

### 附件底图模式流程

1. **用户操作**
   - 启用附件底图
   - 选择附件字段
   - 不选择 base_pdf（可选）
   - 点击"预览 PDF"

2. **后端处理**
   ```
   检测到附件底图模式
   ↓
   base_pdf 为空？
   ↓ 是
   自动设置 base_pdf = "static/blank_a4_landscape.pdf"
   ↓
   打开空白 PDF
   ↓
   下载附件 PDF
   ↓
   将附件页面渲染为高清图片
   ↓
   插入到空白 PDF 页面作为背景
   ↓
   在背景上绘制控件
   ↓
   返回最终 PDF
   ```

3. **最终效果**
   - 附件作为背景
   - 控件绘制在附件上
   - 用户看不到空白 PDF（被附件覆盖）
