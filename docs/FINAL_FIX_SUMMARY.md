### 1. 404 错误修复 ✅

**问题**: 代码在检查文件存在性时就返回 404，没有执行到附件下载逻辑

**修复**: app/routes/designer.py
```python
# 修复前
if not pdf_path.exists():
    return jsonify({"ok": False, "msg": f"底稿不存在: {base_pdf}"}), 404

# 修复后
if not pdf_path.exists():
    # 如果使用附件底图，文件不存在也没关系（会用附件替代）
    if not (cover_source_mode == "attachment" and attachment_bg_enabled and attachment_bg_field_id):
        return jsonify({"ok": False, "msg": f"底稿不存在: {base_pdf}"}), 404
```

**效果**: 现在使用附件底图时，即使 base_pdf 不存在也会继续执行，能看到调试日志了

### 3. 空白模板显示优化 ✅

**修改文件**: app/services/pdf_service.py

**修改内容**:
```python
def list_pdf_templates(cfg: AppConfig) -> List[str]:
    # 空白模板放在最前面
    if filename.startswith("blank_"):
        blank_templates.append(filename)
    else:
        templates.append(filename)
    
    # 排序：空白模板在前，其他模板按名称排序
    return blank_templates + templates
```

**效果**: 
- 空白模板显示在下拉列表最前面
- 方便用户快速找到和选择
