🎯 新功能说明
================================================================================

### 1. 智能参数显示/隐藏

**优化前**: 所有控件显示相同的 30+ 个参数
**优化后**: 每个控件只显示相关参数

示例:
- 文本控件: 显示 15 个相关参数（位置、字体、布局、行为）
- 签名控件: 显示 8 个相关参数（位置、绑定、签名）
- 二维码控件: 显示 6 个相关参数（位置、二维码）
- 矩形控件: 显示 7 个相关参数（位置、形状）
- 线条控件: 显示 8 个相关参数（位置、线条）

### 2. 批量编辑优化

**优化前**: 只能批量编辑完全相同的控件
**优化后**: 可以批量编辑公共参数

#### 同类型控件批量编辑
- 选择多个文本控件 → 可批量编辑所有文本参数
- 选择多个签名控件 → 可批量编辑所有签名参数
- 显示第一个控件的值作为参考
- 显示平均位置和尺寸

#### 不同类型控件批量编辑
- 选择文本 + 签名 + 二维码 → 可批量编辑位置和尺寸 (x, y, w, h)
- 只显示公共参数（基础参数）
- 其他参数自动隐藏

### 3. 用户体验改进

- **友好提示**: "批量编辑模式（已选 3 个 text 控件）"
- **智能禁用**: 唯一性字段（如 field_id）在批量模式下自动禁用
- **控件类型**: 显示控件类型名称，更清晰
- **更新反馈**: "已批量更新 3 个控件的所有参数（15 个参数）"

📊 效果对比
================================================================================

### 单个控件选择

**优化前**:
```
选择文本控件
→ 显示 30+ 个参数（包括签名、二维码等不相关参数）
→ 用户需要手动判断哪些参数有效
```

**优化后**:
```
选择文本控件
→ 只显示 15 个相关参数
→ 界面更清晰，减少混淆
```

### 批量编辑同类型

**优化前**:
```
选择 3 个部门控件（不同 slot_index）
→ 提示"无法批量编辑"
→ 必须一个一个修改
```

**优化后**:
```
选择 3 个部门控件
→ 可以批量修改字体、颜色、对齐等
→ slot_index 等唯一字段保持不变
→ 提示"已批量更新 3 个控件的所有参数（12 个参数）"
```

### 批量编辑不同类型

**优化前**:
```
选择 1 个文本 + 1 个签名 + 1 个二维码
→ 提示"类型不同，无法批量编辑"
→ 无法批量调整位置
```

**优化后**:
```
选择 1 个文本 + 1 个签名 + 1 个二维码
→ 可以批量修改位置和尺寸
→ 只显示公共参数 (x, y, w, h)
→ 提示"已批量更新 3 个控件的公共参数（4 个参数）"
```
