# 文本换行功能使用指南

## 问题诊断

如果文本换行功能不生效，请按以下步骤检查：

## 步骤1：清除浏览器缓存

**方法1：强制刷新**
- Windows/Linux: `Ctrl + Shift + R` 或 `Ctrl + F5`
- Mac: `Cmd + Shift + R`

**方法2：清除缓存**
1. 打开浏览器开发者工具（F12）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

## 步骤2：正确配置文本换行

### 在设计器中操作：

1. **选中部门控件**
   - 点击画布上的部门控件
   - 确保控件被选中（边框变为粉色）

2. **勾选文本换行**
   - 在右侧属性面板中
   - 找到"文本自动换行（text_wrap）- 适用于部门等长文本"
   - ✅ 勾选这个复选框

3. **点击"更新选中项"按钮** ⚠️ 重要！
   - 在属性面板底部
   - 点击蓝色的"更新选中项"按钮
   - 这一步是必须的，否则设置不会保存

4. **验证效果**
   - 在画布中查看控件
   - 如果文本较长，应该能看到换行效果
   - 如果看不到，尝试增大字号测试

5. **保存布局**
   - 点击页面顶部的"保存布局"按钮
   - 确保配置被持久化保存

## 步骤3：测试换行效果

### 测试方法1：在画布中测试

1. 选中部门控件
2. 在属性面板中修改"显示文本（label）"
3. 输入长文本，如：`集团质量管理部一部`
4. 增大字号到 14px 或 16px
5. 点击"更新选中项"
6. 观察画布中的控件是否换行

### 测试方法2：预览 PDF

1. 点击"预览 PDF"按钮
2. 查看生成的 PDF
3. 确认部门文本是否正确换行

## 步骤4：检查配置是否保存

### 方法1：重新选中控件
1. 点击画布空白处取消选择
2. 再次点击部门控件
3. 查看右侧属性面板
4. 确认"文本自动换行"复选框是否仍然勾选

### 方法2：查看保存的配置
1. 点击"保存布局"后
2. 刷新页面
3. 重新选中控件
4. 检查设置是否保留

## 常见问题

### Q1: 勾选了换行但没有效果？

**可能原因：**
- ❌ 没有点击"更新选中项"按钮
- ❌ 浏览器缓存了旧版本的 JS 文件
- ❌ 控件宽度太大，文本没有超出

**解决方法：**
1. 确保点击"更新选中项"
2. 强制刷新浏览器（Ctrl+Shift+R）
3. 缩小控件宽度或增大字号来测试

### Q2: 画布中换行了，但 PDF 中没有？

**可能原因：**
- 没有保存布局配置

**解决方法：**
1. 点击"保存布局"按钮
2. 重新预览 PDF

### Q3: 文本还是撑大了控件宽度？

**可能原因：**
- CSS 样式没有更新

**解决方法：**
1. 清除浏览器缓存
2. 硬性刷新页面
3. 检查控件的 CSS 样式

## 验证 CSS 样式是否生效

1. 打开浏览器开发者工具（F12）
2. 选择"Elements"或"元素"标签
3. 点击画布上的部门控件
4. 在开发者工具中查看该元素的样式
5. 确认以下样式存在：
   ```css
   box-sizing: border-box;
   max-width: 100%;
   word-wrap: break-word;
   white-space: pre-wrap;  /* 启用换行时 */
   word-break: break-all;   /* 启用换行时 */
   ```

## 完整操作流程示例

```
1. 打开设计器页面
2. 强制刷新浏览器（Ctrl+Shift+R）
3. 拖拽"部门控件"到画布
4. 点击选中该控件
5. 在右侧属性面板：
   - 设置宽度：120
   - 设置高度：50
   - 设置字号：14
   - ✅ 勾选"文本自动换行"
6. 点击"更新选中项"按钮 ⚠️
7. 在画布中观察效果
8. 点击"保存布局"
9. 点击"预览 PDF"确认
```

## 调试技巧

### 查看控件的实际配置

在浏览器控制台（F12 -> Console）中输入：

```javascript
// 查看当前选中的控件配置
const selectedIdx = Array.from(state.selected)[0];
const item = state.items[selectedIdx];
console.log('text_wrap:', item.text_wrap);
console.log('完整配置:', item);
```

### 手动设置换行

如果界面设置不生效，可以在控制台手动设置：

```javascript
// 手动启用换行
const selectedIdx = Array.from(state.selected)[0];
state.items[selectedIdx].text_wrap = true;
renderItems();
```
