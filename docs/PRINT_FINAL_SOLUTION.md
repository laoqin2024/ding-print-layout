✅ 打印功能最终修复方案

修复时间: 2026-05-04 17:30
问题: 
1. 打印的是整个页面而不是 PDF 内容
2. 需要自动识别横向/纵向并配置打印纸张方向

================================================================================
🐛 问题分析
================================================================================

问题 1: window.print() 打印整个 HTML 页面
---------------------------------------------------------------
- window.print() 会打印当前页面的所有内容
- 包括导航栏、按钮、背景等
- 即使有 @media print 样式，PDF 嵌入元素也不能很好地打印

问题 2: 无法自动设置打印纸张方向
---------------------------------------------------------------
- 浏览器的 window.print() 无法通过 JavaScript 设置纸张方向
- 需要用户手动在打印对话框中选择横向/纵向
- 钉钉的 dd.biz.util.print() 也不支持设置纸张方向

================================================================================
✅ 解决方案
================================================================================

方案 1: 在钉钉中使用 dd.biz.util.print()
---------------------------------------------------------------
- 使用 GET URL 调用钉钉原生打印
- 钉钉会下载 PDF 并唤起原生打印对话框
- 用户可以在打印对话框中选择纸张方向

方案 2: 在浏览器中打开 PDF 新窗口
---------------------------------------------------------------
- 不使用 window.print() 打印当前页面
- 而是在新窗口打开 PDF
- 用户使用浏览器的 PDF 查看器打印
- PDF 查看器会自动识别纸张方向

方案 3: 使用 @page CSS 规则（部分浏览器支持）
---------------------------------------------------------------
```css
@page {
  size: A4 landscape; /* 或 portrait */
  margin: 0;
}
```
- 但这需要在生成 PDF 时就确定方向
- 而且浏览器支持不一致

✅ 最终实现
================================================================================

修改 1: templates/designer_print.html - 打印样式
---------------------------------------------------------------
```css
@media print { 
  .no-print { display: none !important; }
  body { margin: 0; padding: 0; }
  #pdf-container { 
    width: 100vw !important; 
    height: 100vh !important; 
  }
  #pdf-container iframe {
    width: 100% !important;
    height: 100% !important;
  }
}
```

修改 2: templates/designer_print.html - 打印按钮
---------------------------------------------------------------
```javascript
printFab?.addEventListener('click', () => {
  if (window.dd && dd.env && dd.env.platform !== "notInDingTalk") {
    // 钉钉环境：使用 dd.biz.util.print()
    const pdfUrl = window.location.origin + '/designer/api/preview?' + params.toString();
    dd.biz.util.print({
      url: pdfUrl,
      onSuccess: function () {
        console.log('[DingTalk] 正在唤起钉钉打印...');
      },
      onFail: function (err) {
        // 降级：在新窗口打开 PDF
        window.open(pdfUrl, '_blank');
      },
    });
  } else {
    // 浏览器环境：在新窗口打开 PDF
    const pdfUrl = window.currentPdfUrl;
    if (pdfUrl) {
      window.open(pdfUrl, '_blank');
    } else {
      window.print();
    }
  }
});
```

修改 3: templates/designer_print.html - 使用 iframe 而不是 embed
---------------------------------------------------------------
```javascript
// 使用 iframe 而不是 embed，更好的打印支持
container.innerHTML = `
  <iframe src="${url}" width="100%" height="800px" style="border: none;"></iframe>
`;
```

📊 纸张方向处理
================================================================================

方案: 让 PDF 查看器自动识别
---------------------------------------------------------------
- PDF 文件本身包含页面尺寸信息
- 横向: 842 x 595 (A4 landscape)
- 纵向: 595 x 842 (A4 portrait)

后端生成 PDF 时已经设置了正确的尺寸:
```python
# app/routes/designer.py
desired_orient = "l" if orientation == "l" else "p"
target_w, target_h = ((842.0, 595.0) if desired_orient == "l" else (595.0, 842.0))
doc = fitz.open()
p = doc.new_page(width=target_w, height=target_h)
```

PDF 查看器会自动识别:
- Chrome PDF 查看器：自动识别并建议纸张方向
- Adobe Reader：自动识别并设置纸张方向
- 钉钉打印：自动识别并设置纸张方向
