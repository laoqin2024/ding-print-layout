问题: 无痕模式下参数面板为空，只显示帮助文本

================================================================================
🐛 问题分析
================================================================================

### 问题现象
- 打开设计器页面时，参数面板为空
- 只显示帮助文本（prop-help）
- 所有参数输入框和选择器都不可见

### 问题原因
1. 页面加载时没有选中任何控件
2. `updateParamVisibility` 函数没有被调用
3. 所有参数行保持默认状态（可能被隐藏）
4. 只有帮助文本区域可见

### 根本原因
`updateParamVisibility` 函数负责根据控件类型显示/隐藏参数，但只在以下情况被调用：
- 选中控件时（syncProp 函数中）
- 切换控件类型时

页面初始化时，没有选中任何控件，所以函数没有被调用，导致所有参数都不可见。

================================================================================
🔧 修复方案
================================================================================

### 修改文件
`static/js/designer_edit.js`

### 修改位置
在页面初始化代码中，`renderPropHelp("text")` 之后添加：

```javascript
// 初始化时显示默认控件类型（text）的参数
updateParamVisibility("text");
```

### 修改前
```javascript
renderPropHelp("text");
ensureFixedStatusStamp();
```

### 修改后
```javascript
renderPropHelp("text");

// 初始化时显示默认控件类型（text）的参数
updateParamVisibility("text");

ensureFixedStatusStamp();
```
